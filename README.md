# DionLabs | Signals

The engineering blog for DionLabs.

## Local Development

This project uses `uv` for dependency management.

1. Install dependencies:
   ```bash
   uv sync
   ```

2. Run the development server:
   ```bash
   uv run mkdocs serve
   ```

## Structure

- `docs/`: Markdown source files
- `docs/posts/`: Blog posts organized by year
- `mkdocs.yml`: Configuration for the site
